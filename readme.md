# Desafío 1 – JavaScript

Proyecto básico en JavaScript que calcula el total a pagar de un producto según la cantidad seleccionada.

## Funcionalidad
- Muestra el precio base del producto.
- Permite aumentar y disminuir la cantidad mediante botones.
- Actualiza el total a pagar en tiempo real.
- No permite cantidades negativas.

## Tecnologías
- HTML
- CSS
- JavaScript

## Uso
Abrir el archivo `index.html` en el navegador y utilizar los botones **+** y **-** para modificar la cantidad.

## Notas
Se utiliza el evento `onClick()` para ejecutar funciones de JavaScript desde el HTML y manipular el DOM.
